定义标识符时，必须声明该标识符是变量还是常量
标识符（比如常量名、变量名、函数名）几乎可以使用任何字符
标识符不能以数字开头，不能包含空白字符、制表符、箭头等特殊字符
使用let来定义常量
  let是线程安全的
  只能赋值1次,定义之后不可以修改
  它的值不要求在编译时期确定，但使用之前必须赋值1次
    // 以下写法只有在运行时才知道age的值在swift中也是可以的
    func getAge() -> Int {
        return 10
    }
    let age = getAge()
  如果只是先声明了常量,但是没有指定类型,那么之后赋值的时候会报错,所以如果只是声明一定要加上类型
使用var来定义变量
  定义之后可以修改
声明标识符的格式: 变量/常量关键字 名称 : 数据类型
注意:
  常量、变量在初始化之前，都不能使用
  在真实使用过程中,建议先定义常量,如果需要修改再修改为变量(更加安全)
  是指向的对象不可以再进行修改.但是可以通过指针获得对象后,修改对象内部的属性
